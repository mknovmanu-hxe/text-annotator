# Text Annotation App

Одностраничное приложение (SPA) на **Angular 18+** для создания, редактирования статей и аннотирования текста. Все данные хранятся в `localStorage`.

## 🎯 Функциональность

### Управление статьями (CRUD)

- Создание статьи (plain text, без форматирования и изображений)
- Редактирование статьи
- Удаление статьи
- Просмотр списка статей с возможностью выбора для просмотра/редактирования

### Аннотирование текста

- Выделение произвольного фрагмента текста мышью
- Назначение цвета и текстового примечания (аннотации) выделенному фрагменту
- Подчёркивание аннотированного текста выбранным цветом
- Отображение аннотации при наведении курсора (tooltip)
- Сохранение и восстановление аннотаций из `localStorage`

## ⚙️ Технологии

- **Angular 18+** — основной фреймворк
- **TypeScript** — язык разработки
- **SCSS** — стилизация
- **RxJS** — реактивные подходы
- **localStorage** — эмуляция серверного хранилища
- **Range API** — работа с выделением текста и DOM

> ⚠️ Сторонние библиотеки для UI, аннотаций, хранения данных и тултипов **не используются**. Всё реализовано средствами Angular, HTML, CSS и встроенных API браузера.

## 🚀 Запуск

### Требования

- Node.js 18+
- npm 9+

### Установка и запуск

```bash
git clone https://github.com/mknovmanu-hxe/text-annotator.git
```

```bash
cd text-annotator
```

```bash
npm install
```

```bash
npm start
```
